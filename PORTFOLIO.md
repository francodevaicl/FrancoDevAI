## [P01] Dashboard de Finanzas Personales (versión 0.1)
- Tipo: Análisis de datos + reporte automático en texto
- Problema que resuelve: da una visión clara del total gastado, por categoría y promedios, sin necesidad de que el usuario sepa programar.
- Tecnologías: Python, CSV, procesamiento de datos básico.
- Ruta en repo: `02_data/analisis_gastos.py`
- Archivos generados:
  - `02_data/resumen_gastos.txt` (reporte listo para leer)
- Resultado: informe legible para el usuario con totales, porcentajes y promedios.

## [P02] Mini Dashboard de Gastos (versión 0.1)

- **Tipo:** Análisis de datos + visualización básica + reporte automático
- **Problema que resuelve:** Permite obtener una visión clara del total gastado, promedio, top 3 categorías y gasto máximo, sin necesidad de usar Excel manualmente.
- **Tecnologías:** Python, CSV, Matplotlib
- **Ruta en repo:** `02_data/P02_dashboard.py`
- **Archivos generados automáticamente:**
  - `02_data/grafico_gastos.png` (gráfico de barras por categoría)
  - `02_data/reporte_gastos.txt` (reporte profesional en texto)
- **Resultado:** Dashboard simple que combina procesamiento, análisis y visualización de datos. Primer paso hacia un sistema financiero automatizado.
